# Mentor Activity Agent (Windows)

Локальный агент опрашивает **активное окно** каждые `12` секунд (настраивается), собирает **имя `.exe`** и **заголовок окна**, накапливает длительность и отправляет данные в Laravel: `POST /api/v1/activity/sessions`.

## Сборка

Требуется [Go](https://go.dev/) 1.22+.

```bash
cd activity-agent
go build -o mentor-activity-agent.exe .
```

## Переменные окружения

| Переменная | По умолчанию | Описание |
|------------|----------------|----------|
| `MENTOR_API_URL` | `http://127.0.0.1:8000/api` | Базовый URL API **без** завершающего `/` |
| `MENTOR_TOKEN` | — | **Обязательно.** Bearer-токен Sanctum (`POST /api/auth/login`) |
| `MENTOR_POLL_SECONDS` | `12` | Интервал опроса (секунды или `12s`) |
| `MENTOR_IDLE_SECONDS` | `60` | Если нет ввода дольше этого времени, сессия закрывается |
| `MENTOR_HEARTBEAT_SECONDS` | `60` | Как часто слать `event: heartbeat` по текущей сессии |
| `MENTOR_DEVICE_NAME` | `windows-agent` | Подпись устройства в JSON |

Пример (PowerShell):

```powershell
$env:MENTOR_TOKEN="ваш_токен"
$env:MENTOR_API_URL="http://127.0.0.1:8000/api"
.\mentor-activity-agent.exe
```

## Поведение (точность и границы)

- **Смена окна** — предыдущая сессия завершается, начинается новая.
- **Нет активного окна** (`GetForegroundWindow` = 0), например экран блокировки — сессия завершается.
- **Простой (idle)** — по `GetLastInputInfo`: если пользователь не трогал ввод дольше `MENTOR_IDLE_SECONDS`, сессия завершается; новая не начнётся, пока простой не станет меньше порога.
- **Сон / большой разрыв времени** — если между опросами прошло больше чем в **3 раза** интервал опроса (типично после сна), текущая сессия принудительно закрывается.
- **Офлайн** — неудачные запросы дописываются в `%LOCALAPPDATA%\MentorActivityAgent\pending.jsonl` и дозаливаются при следующих успешных циклах.

Полный контракт API: [docs/activity-api.md](../docs/activity-api.md).

## Автозагрузка

Добавьте ярлык на `mentor-activity-agent.exe` в папку автозагрузки Windows или запланируйте задачу в Планировщике заданий с нужными переменными окружения.
